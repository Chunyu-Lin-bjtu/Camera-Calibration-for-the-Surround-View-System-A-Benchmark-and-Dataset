[2]Kang Liao, Lang Nie, Shujuan Huang, Chunyu Lin, Jing Zhang, Yao Zhao, Moncef Gabbouj, Dacheng Tao, Deep Learning for Camera Calibration and Beyond: A Survey. arxiv preprint arxiv:2303.10559.
